# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-14

### Added
- Initial release
- `CursorTrailProvider` context provider
- `CursorCircularTrail` built-in cursor component
- `CursorTrailArea` polymorphic zone component (div, section, article, etc.)
- `createCursorTrail` factory for custom trail components
- `useCursorTrail` hook for direct cursor state access
- Spring presets: `snappy`, `smooth`, `lazy`
- Full TypeScript types
